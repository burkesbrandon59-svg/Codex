## Termux Metasploit installer

This repository includes `termux-metasploit-installer.sh`, a helper script to install Metasploit inside the Termux app on Android.

### Usage

```bash
chmod +x termux-metasploit-installer.sh
./termux-metasploit-installer.sh
```

The script verifies it is running in Termux on Android, installs required packages, installs `metasploit`, and optionally initializes the Metasploit database.
